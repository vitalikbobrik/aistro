OpenAI Releases GPT-5.5 — Smarter Agents at the Same Speed

Less than two months after shipping GPT-5.4, OpenAI has released GPT-5.5 — its latest and most capable model to date. The April 23, 2026 launch focuses squarely on agentic tasks: coding, computer use, and multi-step research that the model can execute with far less hand-holding than before.

TL;DR: GPT-5.5 is OpenAI's biggest agentic leap yet — same per-token speed as GPT-5.4, but significantly higher intelligence and fewer tokens needed to complete tasks.

What's New in GPT-5.5

Agentic Coding & Computer Use

GPT-5.5 ships with major improvements to Codex — OpenAI's AI coding environment — and to computer use capabilities. The model can now plan, use tools, and verify its own output across multi-step workflows without requiring constant user intervention. OpenAI says it uses significantly fewer tokens to complete the same Codex tasks compared to GPT-5.4, making it both smarter and more cost-efficient for developers.

Deep Research

The model's research mode has been expanded. GPT-5.5 can browse the web, analyze data, write reports, and produce spreadsheets in a single end-to-end session — handling the kind of messy, open-ended work tasks that previously required significant human guidance.

No Speed Trade-off

A common complaint about smarter models is slower output. OpenAI claims GPT-5.5 matches GPT-5.4's per-token latency in real-world serving despite its higher capability level — a significant engineering achievement.

Who Gets Access

GPT-5.5 is rolling out to Plus, Pro, Business, and Enterprise ChatGPT subscribers. It's also available via the API as gpt-5.5 and gpt-5.5-pro. Free users remain on GPT-5.4.

The Bigger Picture

GPT-5.5 arrives just days after DeepSeek V4 and Gemini 3 — a sign that the AI release pace, already relentless in 2025, has only accelerated. For teams already using OpenAI's ecosystem, the upgrade is a straightforward win. For those evaluating alternatives, GPT-5.5 raises the bar that open-source models like DeepSeek need to clear.
